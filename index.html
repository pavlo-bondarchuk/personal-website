3 міні-кейси “Було / Стало”

Швидкість (LCP) Performance
4.8s → 2.2s (-54%)

Було: важкий “hero”, пізнє завантаження критичних стилів і шрифтів.
Стало: оптимізований “hero”, критичний CSS + preload, швидший перший екран.

Стабільність верстки (CLS) UX
0.28 → 0.03 (-89%)

Було: “стрибають” блоки через банери/шрифти/динамічні елементи.
Стало: зафіксовані розміри, правильне підключення шрифтів, прибрані зсуви.

Відгук сайту (INP) Interaction
420ms → 180ms (-57%)

Було: важкі скрипти, зайві обробники подій, блокуючі задачі в main thread.
Стало: розвантажені скрипти, deferred логіка, оптимізація подій і 3rd-party.

Цифри наведені як приклад формату подачі. Під реальні кейси — підставимо ваші “до/після” або посилання на звіти.
